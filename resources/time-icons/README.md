# Time Icons

Download icons for each time period and place them here as SVG files.

## Required Icons

1. `sunrise.svg` - 7am-9am (early morning, sunrise)
2. `morning.svg` - 9am-11am (morning sun)
3. `noon.svg` - 11am-1pm (peak sun, high in sky)
4. `lunch.svg` - 1pm-2pm (lunch break, food/utensils icon)
5. `afternoon.svg` - 2pm-4pm (afternoon sun)
6. `evening.svg` - 4pm-6pm (late afternoon sun, lower)
7. `sunset.svg` - 6pm-8pm (sunset, orange/red)
8. `night.svg` - 8pm-7am (moon, stars, sleep icon)

## Recommended Sources

- **SVG Repo**: https://www.svgrepo.com/svg/384302/day-night-sun-moon-cycle
- **Flaticon**: https://www.flaticon.com/free-icons/day-night-sun-moon (may need attribution)
- **Icons8**: https://icons8.com/icons/set/moon-sun
- **IconScout**: https://iconscout.com/free-icons/sun-moon

## Icon Style Guidelines

- Use consistent style (all same set/theme)
- SVG format preferred (scales better)
- Size: 16x16 or 24x24 pixels
- Simple, clear design
- Monochrome or subtle colors that work with both light/dark themes
